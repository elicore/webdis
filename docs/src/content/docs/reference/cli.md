## Binaries

- `redis-web` (canonical)
- `webdis` (compatibility alias)

## Common commands

```bash
redis-web redis-web.json
redis-web --config redis-web.json
redis-web --write-default-config
```

Alias binary:

```bash
webdis webdis.json
```

The alias is temporary and emits a deprecation message.
