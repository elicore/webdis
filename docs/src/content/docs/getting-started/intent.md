title: Intent
description: Why redis-web exists and what is in scope.
---

`redis-web` is the evolution of this repository beyond strict one-to-one `webdis` implementation goals.

Current intent:

- Keep protocol and config compatibility where it supports migration safety.
- Expose clearer crate boundaries for future surfaces.
- Keep operational setup simple for local and production use.

If you need exact compatibility expectations, read [Webdis Compatibility Scope](/compatibility/webdis-compatibility/).
